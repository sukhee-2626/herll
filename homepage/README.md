# Qlystra Technologies - Homepage Folder

This folder contains the complete EternaCloud homepage design, rebranded for Qlystra Technologies.

## Structure
```
homepage/
├── index.html          # Main homepage (EternaCloud design)
├── css/                # All CSS files
├── js/                 # All JavaScript files
├── images/             # All images (146 files)
├── fonts/              # All font files
└── media/              # All video files
```

## To Run Locally
```bash
# Using Python HTTP server
cd homepage
python -m http.server 8000

# Then open: http://localhost:8000/index.html
```

## Notes
- This is the original EternaCloud "What We Do" page design
- All branding has been changed to "Qlystra Technologies"
- All assets are included
- The current `/index.html` in the root is the "What We Do" subpage

## Deployment
This folder can be deployed as a standalone website to any static hosting service (Vercel, Netlify, GitHub Pages, etc.)
